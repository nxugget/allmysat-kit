# Radio Frequencies

FrequencyFormatter handles standard ITU frequency band classifications.

## Frequency Bands

All bands follow ITU Recommendation M.431-3 nomenclature.

| Band | Range |
|------|-------|
| VLF | 3–30 kHz |
| LF | 30–300 kHz |
| MF | 300 kHz–3 MHz |
| HF | 3–30 MHz |
| VHF | 30–300 MHz |
| UHF | 300 MHz–3 GHz |
| SHF | 3–30 GHz |
| EHF | 30–300 GHz |
